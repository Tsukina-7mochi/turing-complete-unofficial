# Always ON

`#レベル`

## 概要

常にONを出力する部品を作成する問題です (論理式としては恒真式・トートロジーに相当します)。

## 攻略

どのような入力に対しても常に <span class="T">ON</span> を出力させます。
入力と繋いだときに常に <span class="T">ON</span> または <span class="F">OFF</span> となるゲートはあるでしょうか？
